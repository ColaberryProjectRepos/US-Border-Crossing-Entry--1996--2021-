Inserting the Year Slicer visual 12

1-Select the Slicer from the Visual

2 Select the Year from the Date Dim

  Go to the General tab and adjust the size X-Position 987, Y-Position 0, Width 191, and Height 106

4- Turn off the Slicer Header 

5- Expand Numeric inputs and Font Color  #000000,  Text size 10, Font family Segoe UI

6- Turn on the Title write Year, Font Color  #000000,  Background Color #FFFFFF, Alignment Center, Text size 16, Font family DIN.

7- Turn on the Border and change the color to #E6E6E6.
